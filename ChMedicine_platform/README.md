# 中医诊疗平台

#### 介绍
中医诊疗平台是一个在线的医务平台，中医诊疗平台为中医诊疗提供了新的解决方案。1.0版本实现了电子处方，处方比对，药品库存管理，诊疗记录库存管理，名医医案库存管理五大核心模块。平台采用了layui+springboot+mysql模式开发，易于作为二次开发的基础系统,自适应界面。
